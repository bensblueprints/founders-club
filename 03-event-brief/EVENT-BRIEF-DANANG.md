## Format (the arc of the night)
1. **App access** — everyone's profile is live; people already know who's in the room.
2. **Host-led bonding + introduction activities** — structured icebreakers so nobody's stranded; each person gives a short self-intro/presentation.
3. **Free time** — dinner, move between tables, catch up organically. This is where the real connecting happens.

## Digital layer on the night (see full spec)
See [Web App Spec](../05-web-app/WEB-APP-SPEC.md). On the night it should let an attendee:

- Get suggested matches ("you should meet ___ because ___").

## Language (hard requirement)
Prior events ran ~50% Vietnamese / 50% international, with many VN attendees not comfortable in English.
- [ ] **Host/MC must be bilingual (Vietnamese + English)** and run key moments in both.
- [ ] Name tags / signage / app in both languages where practical.
- [ ] Seating/intros consider language comfort so no one is stranded.
- [ ] **Pre-event 1:1 support for VN attendees not comfortable in English** — reach out directly, give guidance on making the most of the event, and help them prepare their short English self-intro/presentation so they're confident on the night.
