# Founders VN Logo Kit

Use these assets for webpages, slides, and sharing.

## Recommended

- `founders-vn-logo.svg` - Full Founders VN logo with transparent background.
- `founders-vn-wavy-white.svg` - Wavy mark only, white, transparent background.
- `founders-vn-wavy-dark.svg` - Wavy mark only, dark green, transparent background.
- `founders-vn-wavy-icon.svg` - Wavy mark on dark rounded-square icon.

## PNG fallback

- `founders-vn-logo.png`
- `founders-vn-wavy-white.png`
- `founders-vn-wavy-dark.png`
- `founders-vn-wavy-icon.png`

## Vector trace alternates

These are pure SVG path traces. They are smaller, but the source image was low resolution, so the high-resolution SVG files above usually look better in design tools.

- `founders-vn-wavy-vector-white.svg`
- `founders-vn-wavy-vector-dark.svg`
